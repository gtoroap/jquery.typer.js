jquery.typer.js

A jQuery plugin to display text as if was typewriter.

Getting Started

Include jQuery and the plugin on a page. Then call the typer method on selector with text (prefered paragraph and span).

<script src="jquery.js"></script>
<script src="jquery.typer.js"></script>
<script>
  $("selector").typer();
</script>
